This is a template for an academic personal website. It was created with grad students and postdocs in mind, however, others may find it a helpful template as well! We’re hopeful that it will be relevant to those seeking industry as well as academic positions and have created page options that we think are more industry-relevant, however we are more familiar with academic websites. The github repository is https://github.com/ngalanter/academic-website-template.

This template uses quarto, you can learn more about quarto websites here: https://quarto.org/docs/websites.

This page uses the about page template, see details here: https://quarto.org/docs/websites/website-about.html. This page can be used to give an overview of yourself, and you can link externally and internally.

This template uses the minty theme with slight color customization, to change the theme or create a custom theme go to the _quarto.yml file. See here for more details about themes and see here for a preview of what the pre-made themes generally look like.

See the repository readme for more information about using this template and publishing via quarto and github pages.

This goal of this template is to provide several types of pages that might be relevant, but most personal websites don’t have all of these pages!
